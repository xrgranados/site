# Instagram jQuery plugin that uses Slimbox2 to display the standard resolution file.


## Usage

1
----------------------------------------------- */

Register with Instagram using the following link:

http://instagram.com/developer/

This will give you your Client ID.


2
----------------------------------------------- */

Open Index.html and replace the words "ENTER YOUR CLIENT ID HERE" with your CLIENT ID.


3
----------------------------------------------- */

Still in Index.html replace the words "ENTER YOUR HASHTAG HERE" with your relevant hashtag. Note you DON'T need the #. Just enter the word.


4
----------------------------------------------- */
Save and open in a browser. Modify the CSS and HTML markup as you please.
